# blogs

This repo is for blog source code.

## Tools require

| tool   | version |
|--------|---------|
| Java   | 17+     |
| Maven  | 3.8+    |
| Spring | 3.0.0+  |

## How to get code.

```shell
git clone git@github.com:yoa1226/blogs.git
```

## Build and Run test

build and run test whole project

```shell
mvn clean package
```

build and run test specify module

```shell
mvn test -Dtest=testname -pl subproject
```
